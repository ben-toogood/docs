### Supported Services

Pimento Supports the following Evri services:

- Evri Next Day
- Evri Standard

### Connecting Evri

In order to connect your Evri account to Pimento, we require the following information:

- Username
- Password
- Client ID
- Client Name
- Tracking API Key
- Tracking Client ID
- Tracking Client Secret
