---
title: Triggers
---

Triggers are the start of a workflow. It can be an order being created or an issue shipped. From your selected trigger point there will be a whole range of checks and actions populated to choose from.

Options to choose from:

- **Order cancelled**: Order status updated to cancelled

- **Order created**: Order created in Pimento

- **Order placed on hold**: Order status updated to on hold

- **Order Shipped**: Order Status updated to shipped

- **Issue approved**: An issue was approved (A replacement will be dispatched)

- **Issue rejected**: An issue has been rejected (No replacement will be dispatched)

- **Issue Reported**: An issue was reported

- **Issue Resolved**: An issue was resolved (A replacement has been dispatched)
