title: Subbly
---

To connect your Subbly account to HutchOS, follow the below steps:

1. Navigate to the **Admin** section of the Subbly dashboard. Select **Storefont API Keys** from the sidebar.

<img height="200" src="/images/subbly-1.png" alt="Step 1" />

2. Press **Generate API Key** and copy the new API key which will be generated

3. Once you have these credentials, navigate to [Settings > Integrations](https://web.tryhutch.co.uk/clients/settings/integrations) within the HutchOS dashboard, and press **Connect** within the Subbly section.

<img height="200" src="/images/subbly-2.png" alt="Step 2" />

4. Enter your API Key in the popup and then press **Connect**. Your Subbly store is now connected to HutchOS.

<img height="200" src="/images/subbly-3.png" alt="Step 3" />
